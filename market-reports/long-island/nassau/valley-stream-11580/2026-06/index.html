How this number is calculated The Zillow Home Value Index (ZHVI) is a smoothed, seasonally-adjusted measure of the typical home value across all home types within ZIP code 11580. Zillow's methodology aggregates millions of public records, Multiple Listing Service feeds, and proprietary signals into a single value per ZIP per month. Unlike a simple median sale price, ZHVI accounts for compositional mix — so a month with mostly high-end sales doesn't artificially inflate the index. Source: Zillow Research methodology.
